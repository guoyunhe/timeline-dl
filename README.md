# timeline-dl

## Install

### Run directly

```bash
npx timeline-dl
```

### Install on system

```bash
npm i -g timeline-dl
%binName%
```

### Install in project

```bash
npm i -D timeline-dl
```

Add script entry:

```json
{
  "scripts": {
    "%binName%": "%binName%"
  }
}
```

Run:

```bash
npm run %binName%
```

## Options

### `--help`

Show help

### `--version`

Show version
